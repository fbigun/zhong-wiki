## 基础语法 ##

单行注释
```
// 单行注释
```

多行注释
```
/*
  多行注释
*/
```

定义变量
```
var myVariable = "something"
```

调用变量
```
myVariable
```

定义函数
```
function doSomething() {
  alert( "Function" );
}
```

调用函数
```
doSomething();
```

## 数据类型 ##

字符串(string)：JavaScript解释器要求字符串数据被单引号或双引号括起来。
```
document.write( "some characters" );
```

数字(number)：一个数，包括浮点数。

布尔值(boolean)：包含一个真值(true)或假值(false)。

空值(null)：表示没有数据。

未定义(undefined)：表示没有定义也没有赋值。使用变量，这种类型十分重要。
